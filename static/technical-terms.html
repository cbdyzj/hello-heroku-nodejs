Technical Terms

名字 说明
C 简单而锋利的C语言，历久弥新的通用编程语言，计算机世界的基石
Linux 自由、开源的类UNIX操作系统
JavaScript 世界上最好的编程语言，浏览器钦定的脚本语言
macOS 苹果电脑的操作系统，漂亮的macOS谁人不爱呢？
Android 安卓操作系统，绿色的小机器人
Java 群众基础很好的编程语言，据说有30亿设备在运行Java
常用于开发后台程序和安卓应用
Ruby 小众编程语言，有著名的Ruby on Rails框架
Kotlin JVM的方言，现已经被谷歌钦点为安卓推荐开发语言
Scala JVM的方言，很小众，很难学；在数据处理领域有运用
Python 常见编程语言，在人工智能领域有运用
PHP 用PHP可以开发网站
C++ 通用程序设计语言，支持多种编程范式
Rust 精心设计的内存安全的通用程序设计语言
Go Go语言通常用来开发后台程序，提供了内建的并发模型
C# 和Java差不多但是用的人少很多
TypeScript JavaScript的方言，为JavaScript增加了类型标注
React 前端框架
Vue.js 前端框架
Angular 前端框架
Git 分布式版本控制软件
GitHub 通过Git进行版本控制的软件源代码托管服务平台
Ruby on Rails Ruby语言开发的开源Web应用框架
Gradle 自动化建构工具
Apache Maven 自动构建工具
webpack 前端打包工具
Node.js Node.js是能够在服务器端运行JavaScript的开放源代码、跨平台运行环境
Nginx Nginx高性能的网页服务器，也可以用作反向代理、负载平衡器和HTTP缓存
MySQL 常用的开源关系数据库系统
RabbitMQ 常用的开源消息队列
Elasticsearch 常用的开源搜索引擎
Redis 常用的内存数据库
ClickHouse ClickHouse是一个用于联机分析处理（OLAP）的开源列式数据库
Cassandra Apache Cassandra（简称为C*）是一套开源分布式NoSQL数据库系统
Spring Framework 广受欢迎的Java平台的开源应用开发框架
.NET Framework 微软开发，一个致力于敏捷软件开发、快速应用开发、平台无关性和网络透明化的软件框架
Electron Electron使用Node.js（作为后端）和Chromium的渲染引擎（作为前端）完成跨平台的桌面GUI应用程序的开发
